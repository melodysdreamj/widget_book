# Widget Book
1. add any view lego
```bash
lego add [lego-name]
```

2. generate widget book
```bash
dart run lib/auto_script.dart
```

3. build flutter web
```bash
flutter build web
```

4. deploy flutter web to cloudflare
```bash
wrangler publish
```
